bean
====

Bean Stack - Bootstrap CSS, ExpressJS, AngularJS, NodeJS
====

Simply clone to repo.

Then run npm install.

Once Jade, Less-Middleware and Express are installed, you can run "node app.js" and should be able to view the server on localhost:3000.
